Statistics Regarding COVID-19 in India


Cumulative Confirmed Cases ⇉

The data for this graph has been obtained with the aid of MongoDB COVID-19 Open Data project, which is a free cluster for data on COVID-19, that is sourced from John Hopkins CSSE. The data is updated everyday, and is built for easy integration. We have used the requests library in Python to visualise the data!



⇇ Stats for Cases observed daily

Python helps a lot with converting data into a graphical, user-friendly format! It has simple and clean syntax, object encapsulation, good library support, and optional named parameters. It may be hard initially to adapt to the indentation rules, but when it all works together in a project, it makes it feel worth it!
